# Steps

`Steps` is a navigation bar that guides users through the steps of a task.


## When To Use

When the task is complicated or has a certain sequence in the series of subtasks, we can decompose it into several steps to make things easier.

### Switch Step

<Playground>
  <Value initial={0}>
    {({ value, set }) => {
      const steps = [{
        title: 'First',
        content: 'First-content',
      }, {
        title: 'Second',
        content: 'Second-content',
      }, {
        title: 'Last',
        content: 'Last-content',
      }];

      return (
        <div>
          <Steps current={value} onCurrentChange={set}>
            {steps.map(({ title }) => (
              <Steps.Step key={title} title={title} description="This is a description" />
            ))}
          </Steps>

          <Flex
            my="4"
            height="200px"
            alignItems="center"
            justifyContent="center"
            bg="gray300"
            borderRadius="1"
          >
            {steps[value].content}
          </Flex>

          {value < steps.length - 1 && (
            <Button type="primary" onClick={() => set(value + 1)}>Next</Button>
          )}
          {value === steps.length - 1 && (
            <Button
              type="primary"
              onClick={() => message.success('Processing complete!')}
            >
              Done
            </Button>
          )}
          { value > 0 && (
            <Button ml="2" onClick={() => set(value - 1)}>
              Previous
            </Button>
          )}
        </div>
      );
    }}
  </Value>
</Playground>
