# Dropdown

A dropdown list.

## When To Use

If there are too many operations to display, you can wrap them in a `Dropdown`. By clicking/hovering on the trigger, a dropdown menu should appear, which allows you to choose one option and execute relevant actions.

## Examples

```js
import { Dropdown } from '@yoctol/ui';
```

### SubItem

<Playground>
  <Dropdown
    placement="bottomLeft"

      <Dropdown.List textAlign="center">

        <Dropdown.Item key="Item 2">Item 2</Dropdown.Item>
        <Dropdown.SubList key="Item 3" title="Sub List">
          <Dropdown.Item key="1" onClick={() => console.log('sub item 1')}>
            Sub Item
          </Dropdown.Item>
          <Dropdown.SubList
            key="Overflow Sub List"
            title="Overflow Sub List"
            maxHeight="200px"
            overflow="auto"
          >
            {range(1, 20).map(key => (
              <Dropdown.Item
                key={key}
                onClick={() => console.log('sub list item 1')}
              >
                Sub List {key}
              </Dropdown.Item>
            ))}
          </Dropdown.SubList>
        </Dropdown.SubList>
      </Dropdown.List>
    }
  >
    {({ visible, toggle }) => (
