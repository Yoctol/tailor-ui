# TextField

Text fields let users enter and edit text.

## When to use

When a maxLength and label needs to be provided.

## Examples

```js
import { TextField } from '@yoctol/ui';
```

<Playground>
  <TextField label="Text Field" onChange={event => console.log(event.target.value)} />

</Playground>

### With textarea & error

<Playground>
  <Input>
    {({ value, bind }) => {
      const hasError = value === 'error';
      return (
        <TextField
          textarea
          minRows={2}
          maxRows={6}
          label="Please type 'error'"
          error={hasError}
          message="Just show error"
          {...bind}
        />
      );
    }}
  </Input>
</Playground>
