## Examples

First, install it:

```bash
yarn add @yoctol/ui
```

And whatever you want, just import (Remember, the component should wrap with `UIProvider`):

```js
import { UIProvider, Button } from '@yoctol/ui';
```

```jsx
<UIProvider>
  <Button>OK</Button>
</UIProvider>;
```

## Development

Clone it:

```bash
git clone git@github.com:Yoctol/ui.git
```

Instal dependencies:

```bash
cd ui && yarn
```

Start the docz:

```bash
yarn docz:dev
```
