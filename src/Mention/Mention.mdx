# Mention

Mention component.

## When To Use

When need to mention someone or something.

## Examples

```js
import { Mention } from 'tailor-ui';
```

### Basic

<Playground>
  {() => {
    const [value, setValue] = React.useState(Mention.toContentState('我想要一杯 @飲料'));

    return (
      <Mention
        autoFocus
        suggestions={['飲料', '食物']}
        value={value}
        onChange={contentState => {
          setValue(contentState);
          console.log('Raw value: ', Mention.toString(contentState));
          console.log('Mentions: ', Mention.getMentions(contentState));
        }}
      />
    );

}}

</Playground>
