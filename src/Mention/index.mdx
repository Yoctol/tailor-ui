# Mention

Mention component.

## When To Use

When need to mention someone or something.

## Examples

```js
import { Mention } from '@yoctol/ui';
```

### Basic

<Playground>
  <Value initial={Mention.toContentState('我想要一杯 @飲料')}>
    {({ value, set }) => (
      <Mention
        suggestions={['飲料', '食物']}
        value={value}
        onChange={contentState => {
          set(contentState);
          console.log('Raw value: ', Mention.toString(contentState));
          console.log('Mentions: ', Mention.getMentions(contentState));
        }}
      />
    )}
  </Value>
</Playground>
