# Table

A table displays rows of data.

## When To Use
- To display a collection of structured data.
- To sort, search, paginate, filter data.

## Examples

```js
import { Table } from '@yoctol/ui';
```

### Basic Table

<Playground>
  <Table>
    <Table.Head>
      {range(2, 10).map(x => <Table.HeadColumn key={x}>{x}</Table.HeadColumn>)}
    </Table.Head>
    <Table.Body>
      {range(1, 10).map(x => (
        <Table.Row key={x}>
          {range(2, 10).map(y => (
            <Table.Column key={y}>
              {y} * {x} = {x * y}
            </Table.Column>
          ))}
        </Table.Row>
      ))}
    </Table.Body>
  </Table>
</Playground>
