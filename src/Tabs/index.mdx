# Tabs

Tabs make it easy to switch between different views.

## When To Use

Use tabs for functional aspects of a page.

## Examples

```js
import { Tabs } from 'tailor-ui';
```

### Basic

<Playground>
  {() => {
    const [value, setValue] = React.useState('1');

    return (
      <>
        <Tabs activeValue={value} onChange={setValue}>
          <Tabs.Tab value="1" label="Tab 1" />
          <Tabs.Tab value="2" label="Tab 2" />
          <Tabs.Tab value="3" label="Tab 3" />
        </Tabs>
        <p>Active Tab is {value}</p>
      </>
    );

}}

</Playground>
