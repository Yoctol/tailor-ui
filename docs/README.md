### [Docz](https://yoctol-ui.netlify.com)

### Examples

First, install it:

```bash
yarn add @yoctol/ui
```

And whatever you want, just import:

```js
import { Button } from '@yoctol/ui';
```

### Development

Clone it:

```bash
git clone git@github.com:Yoctol/ui.git
```

Instal dependencies:

```bash
cd ui && yarn
```

Start the docz:

```bash
yarn docz:dev
```
